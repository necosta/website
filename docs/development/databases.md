# Databases

Breakdown of database types and functionality

### What type of databases are available?

- RDBMS (example: `PostgreSQL`)
- No-SQL (example: `DynamoDB`)
- Memory (example: `Redis`)
- Graph (example: `Neo4J`)

## Aspects

- Indexing
- Transactions
- Schema design
- Query optimizations
